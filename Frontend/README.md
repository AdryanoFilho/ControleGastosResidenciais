# Frontend — Controle de Gastos Residenciais

Interface web em **React + TypeScript (Vite)**.

```bash
npm install
npm run dev
```

Disponível em `http://localhost:5173` (requer a API em execução — ver instruções completas no [README na raiz do repositório](../README.md)).

## Scripts

| Comando | Descrição |
|---|---|
| `npm run dev` | Servidor de desenvolvimento |
| `npm run build` | Verificação de tipos + build de produção |
| `npm run lint` | Análise estática (oxlint) |

## Configuração

| Variável | Padrão | Descrição |
|---|---|---|
| `VITE_API_URL` | `http://localhost:5000/api` | URL base da API |
